Use Open Source DevOps tools with VMware vCloud Air



The Demo Environment

This is a live sandbox demonstrating open source DevOps tools interacting with vCloud Air.

You need a login to run the demos, but all servers are configured so anyone can view previous executions.





Infrastructure Automation Examples

These scenarios use automation tools to create Virtual Machines on vCloud Air

Scenario Description
Photon Server Example Create a Photon Server on vCloud Air using vCloud Air CLI
Docker / Photon Example Deploy a docker container on a Photon Server
Self Provision Corp Std Web Server Use chef to configure a Server as a standard Web Server
Chef Knife Plugin Create an Ubuntu Server on vCloud Air using the Chef vCloud Air Knife plugin
Ansible Client outside vCloud Air Provision a CentOS server on vCloud air and deploy an application to nginx.
