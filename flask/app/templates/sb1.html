SB1 Scenario

This scenario demonstrates an continuous integration build pipeline using GitHub, Jenkins and Chef.
A Spring Boot application is built and deployed whenever a developer modifies the code in the GitHub Project

The sequence is as follows...

  1. The developer checks in a modification to GitHub
  2. GitHub notifies Jenkins of the change
  3. Jenkins retrieves the code from Github
  4. Jenkins instructs Gradle to build the project
  5. Jenkins instructs Chef to provision the VM for the application
  6. Jenkins deploys the new version of the Application
  7. The developer accesses the new version of the App
