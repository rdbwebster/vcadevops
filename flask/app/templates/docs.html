Installing vca-cli
The easiest way for a developer to work with vCloud Air is using the vCloud Air CLI.
The instructions in the documents below will use cli commands to complete the steps.
vca-cli is an open source project. Further installation instructions and documentation can be found on github https://github.com/vmware/vca-cli
The same actions can be performed using the vCloud Air Portal if desired.
